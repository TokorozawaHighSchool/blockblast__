# Tetris Project

This project is a simple implementation of the classic game Tetris using HTML, CSS, and JavaScript. The game allows players to control falling blocks and complete lines to score points.

## Project Structure

The project consists of the following files:

- `src/index.html`: Defines the HTML structure of the Tetris game, including the game canvas and start button.
- `src/main.js`: Implements the game logic, including block generation, movement, rotation, collision detection, and line clearing.
- `src/styles.css`: Contains the styles for the game, including the appearance of the canvas and blocks, as well as animations.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `src/index.html` file in a web browser to start playing the game.

## How to Play

- Use the arrow keys to move and rotate the falling blocks.
- Complete a line to clear it and score points.
- The game ends when the blocks reach the top of the canvas.

Enjoy playing Tetris!
